## 🌟 Project Overview
This repository contains the source code for a multi-page static corporate website designed for **BuildCraft Construction Pvt. Ltd.**, a premier construction company. The website showcases the company's services, project portfolio, team members, and contact information. The goal of this project was to design a fully functional, easy-to-navigate website using HTML that effectively represents a professional business.

## 📂 File Structure & Pages

The website is composed of five main HTML pages, linked together via a consistent navigation menu:

* **`index.html` (Home Page):** * Serves as the landing page.
    * Introduces the company, highlights reasons to choose BuildCraft, and provides a quick overview of services and featured projects.
* **`about.html` (About Us):** * Details the company's history (established in 2010), mission, vision, and core values.
    * Features a "Meet Our Experts" section introducing the CEO, Chief Architect, Project Manager, and other key team members.
    * Lists major company achievements.
* **`services.html` (Services):** * Provides in-depth descriptions of 8 core services (Residential, Commercial, Industrial, Renovation, Structural Engineering, Road & Infrastructure, Project Consultancy, and Turnkey Projects).
    * Includes a Project Estimation & Timeline Guide table and a detailed project consultation request form.
* **`gallery.html` (Gallery/Portfolio):** * Displays a structured showcase of featured completed projects, including a Luxury Villa, Apex Commercial Tower, Global Logistics Hub, and a Metro Flyover.
    * Includes project statistics and an FAQ section regarding site visits and blueprints.
* **`contact.html` (Contact Us):** * Features comprehensive contact information including phone, email, and office hours.
    * Integrates an interactive Google Maps iframe for the office location.
    * Contains a functional message submission form with radio buttons and text areas.

## ⚙️ Features & Functionalities
* **Consistent Navigation:** Every page includes a `<nav>` bar making it easy to jump between Home, About, Services, Gallery, and Contact pages.
* **Forms & User Input:** Interactive web forms on the Services and Contact pages utilizing various input types (text, email, tel, radio buttons, checkboxes, textareas).
* **Media Integration:** Use of images and iframes (maps) to enhance the visual experience.
* **Semantic Data Organization:** Extensive use of HTML tables, lists (unordered and ordered), and structural tags to display data like project timelines and team roles cleanly.

## 💻 Technologies Used
* **HTML5:** Core markup language used for structuring all web pages and embedding media/forms.

## 🚀 How to Run the Project
1. Clone or download this repository to your local machine.
2. Ensure all HTML files and images are kept in their respective directory paths.
3. Open `index.html` in any modern web browser (Google Chrome, Mozilla Firefox, Safari, Edge) to view the website.
